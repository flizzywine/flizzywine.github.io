# 周易网站重写任务

## 项目概述
将现有的周易查询工具网站（index.html）进行现代化重写，保持所有功能不变。

## 现有功能
1. **易经卦爻查询**: 输入上卦(1-8)、下卦(1-8)、动爻(1-6)查询卦象
2. **小六壬计算器**: 输入三个数字计算小六壬结果
3. 数据来源: data.json（包含64卦完整数据）

## 技术要求
- 使用现代前端技术（HTML5, CSS3, JavaScript）
- 响应式设计，支持移动端
- 美观的 UI，符合中国传统文化风格
- 保持原有功能完整
- 添加适当的动画效果
- 优化用户体验

## 输出要求
1. 重写 index.html
2. 创建现代化的 CSS 样式
3. 优化 JavaScript 代码结构
4. 编写详细的 README.md

## 数据文件
- data.json: 包含64卦的完整数据（卦辞、彖传、大象传、爻辞等）

请分析现有代码，然后进行重写。
